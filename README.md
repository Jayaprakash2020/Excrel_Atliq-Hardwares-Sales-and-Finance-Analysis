#  AtliQ Hardware Sales & Finance Analysis


##  Project Overview  
 - This project analyzes the **sales and financial performance of AtliQ Hardware** using Microsoft Excel.  
 - It combines **Sales Analytics** and **Finance Reporting** to generate insights that support business decision-making.


##  Project Objectives  

### Sales Analysis  
- Create a Customer Performance Report
- Compare Market Performance vs Sales Targets

### Finance Analysis  
- Generate Profit & Loss (P&L) reports by:
  - Fiscal Year  
  - Monthly performance  
  - Market-wise analysis  

## Dataset Description  
The dataset includes:
- Customer-level sales data  
- Year-wise revenue (2019–2021)  
- Market and regional performance  
- Financial metrics (P&L)
## Key Insights  

###  Sales Growth  
- 2019 → **87.5M**  
- 2020 → **196.7M**  
- 2021 → **598.9M**  
- Growth Rate → **304.5% (2020–2021)**  

### Top Performing Customers  
- Amazon  
- AtliQ e Store  
- AtliQ E Exclusive  

## Reports Included  

### Sales Reports  
- Customer Performance Report  
- Market vs Sales Target Analysis  

###  Finance Reports  
- P&L by Fiscal Year  
- P&L by Month  
- P&L by Market  

##  Technical Skills  

-  ETL Process (Extract, Transform, Load)  
-  Power Query for data transformation  
-  Data modeling used develop the relationship with tables
-  Using Power Pivot to organize and categorize the data
-  Creating calculated columns using DAX  
-  Time-based analysis (Fiscal months & quarters)
-  Using conditional formating to enhance the report

##  Soft Skills  

-  Business understanding about sales and finance
-  Identify patterns (top customers, seasonal sales)
-  Analytical thinking and problem-solving  
-  Structured approach to report development
-  Creating a user-friendly report

##  Conclusion  
This project demonstrates how Excel can be used to perform **end-to-end data analysis**, from data preparation to insight generation.
